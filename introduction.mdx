# Welcome to SANL

SANL (Sistema De Atención Nutricional En Línea) is a modern digital health platform designed to transform nutritional monitoring and patient care management. Built with cutting-edge technologies, SANL empowers healthcare professionals to deliver efficient, data-driven nutritional care.

## What is SANL?

SANL is a comprehensive digital health solution that bridges the gap between traditional clinical record-keeping and modern AI-powered insights. The platform enables nutritionists and healthcare providers to:

- Digitize and manage clinical nutritional records
- Generate AI-powered insights and recommendations
- Create professional PDF reports for patients
- Access real-time data from anywhere

## Technology Stack

SANL is built using modern, production-ready technologies:

### Frontend
- **Next.js 16.1.6** - React framework with server-side rendering and optimal performance
- **React 19** - Latest React features for building interactive UIs
- **Tailwind CSS 4** - Utility-first CSS framework for rapid UI development
- **TypeScript** - Type-safe development for robust code

### Backend
- **FastAPI** - High-performance Python web framework for building APIs
- **Uvicorn** - Lightning-fast ASGI server
- **Google Generative AI** - Integration with Gemini 1.5 Flash for AI capabilities
- **Supabase** - PostgreSQL database with built-in authentication and real-time features
- **FPDF2** - PDF generation library for creating professional reports

## Architecture Overview

SANL follows a serverless architecture pattern with clear separation of concerns:

```
SANL/
├── frontend/          # Next.js application with TypeScript
├── backend/           # FastAPI serverless functions
├── docs/              # Documentation (you are here!)
└── analysis/          # Project analysis and planning
```

<Info>
SANL is designed with scalability and maintainability in mind, following modern software development best practices and serverless architecture principles.
</Info>

## Key Features

### Digital-First Approach
Transform your nutritional practice from paper-based to fully digital, with all patient data securely stored and easily accessible.

### AI Integration
Harness the power of Google's Gemini 1.5 Flash model to provide intelligent nutritional insights and recommendations based on patient data.

### Professional Reporting
Generate comprehensive, branded PDF reports that you can share directly with patients or print for physical copies.

### Real-Time Sync
With Supabase as the backend, all data is synchronized in real-time, ensuring you always have the latest patient information.
